# 需求文档

## 目的

网站作为本人业务能力、专业能力和有趣性格的名片，同时建立一个客户联系渠道。

【业务能力】：展示已完成的项目和过往的工作经历。

【专业能力】：
1. 创意能力: 展示本网站是如何通过 trendy 的技术栈完成的，展示某些技术细节，增加能力的 credibility。
2. 全栈能力：
2.1 前端：a11y, 中英文切换，黑暗模式切换
2.2 后端：API integration, Next.js 功能，SEO

【有趣的灵魂】：
同时网站上有独创性的动画或者功能，展示有趣的生动的 personality 和活人感。

## 功能

### 导航

功能：
1. 展示个人 logo, 需要适当设计动画和样式，展示有趣的灵魂
2. 子页面导航
3. 黑暗模式切换
4. 中英文切换

### 子页面

#### 首页

1. hero section: 一句话的个人介绍，突出全栈工程师和 freelancer 两个重点。
2. 重点项目卡片
3. skill set 展示
4. 立即联系的 CTA

#### 项目

以卡片的形式组织每个项目的信息：
- 图片
- description
- 技术栈
- github 链接（如有）
- 网站链接（如有）
- 花费的时间：说明这个项目花费了多少时间

#### 联系页面

联系页面主要承载两个功能：联系渠道和个性化展示

1. 个人头像
2. 当前时间是否可以工作
3. 真实的客户评价展示
4. 更多个人资料跳转：github 主页, linkin, instagram
5. upwork 入口，Fiverr 入口，email 入口

### Develop Mode 开发者模式

1. Tailwind：开启后，hover 展示元素的 classname
2. React: 开启后，hover 展示组件的 component name, 和 props 及其类型
3. Documentation: 跳转到 Google Drive 链接

### 设计风格
设计风格关键词：
- 浅灰背景、低饱和度粉色字体和按钮
- 极简、现代、高级感
- 干净留白、清晰排版
