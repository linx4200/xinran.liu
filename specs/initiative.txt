我想要创建一个个人网站。

1. 网站目标
1.1 展示项目与技能;
1.2 网站本身作为展示技术能力的项目;
1.3 为潜在客户提供了解和联系我的入口，提高转化率。

2. 页面结构（共 5 个页面）
2.1. 首页 /           — 精简、聚焦，突出身份、技术栈、行动按钮
2.2. About Me /about   — 展示经历、技术时间线、理念、技能图谱
2.3. Projects /projects — 项目卡片 + 项目详情（动态路由）
2.4. Blog /blog         — 技术博客，使用 markdown 渲染
2.5. Hire Me /hire       — 联系表单、合作流程、客户评价等

3.设计风格关键词：
- 浅灰背景、低饱和度粉色字体和按钮
- 极简、现代、高级感
- 干净留白、清晰排版

4. 技术栈：
- Next.js
- React
- Tailwind CSS
- 工程能力：动效， a11y, SEO

5. 网站亮点：
- 网站项目本身作为一份作品集交互式源码
- Tailwind CSS 类名可视化设计：让访问者感受到你对 Tailwind 的掌握和使用哲学
- Next.js 特性展示：通过网站行为展示你对 Next 的掌握（SSR、API Routes、Routing、ISR 等）， 例如：
- - 点击某个 UI 元素时加载数据
- - /api 路由可浏览
- - Router 可视化
-  Dev 模式（开发者视图）： 吸引技术观众，作为你的网站「隐藏彩蛋」
